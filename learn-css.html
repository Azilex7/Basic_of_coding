Apprendre le CSS

CSS est l'abréviation de Cascading Style Sheets (feuilles de style en cascade). Il a été créé dans les années 90. Son but est d'ajouter du style aux balises HTML. Avec CSS, vous pouvez changer les couleurs, les tailles et même créer des animations ! même créer des animations !

hi{
color: #fff;
font-size: 25px;
font-weight: 700;
}

Deux outils sont très importants en CSS : flex et grid.
