Apprendre le HTML

HTML est l'abréviation de Hyper Text Markup Language (langage de balisage hypertexte). Il permet de décrire la structure d'une page web.

<!DOCTYPE html>
<html lang="en">
<head> <meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Mon titre</title>
</head>
<body>
</body>
</html>

Voici à quoi HTML ressemble. C'est impornat de noter que le code se divise en deux grandes parties
-<head> : C'est ici que sont noté les métadonées de la page web. Dedans, on peut y ajouter par exemple le titre de la page, où trouver le style, etc.
-<body> : C'est le contenue de votre page web.
Quelques exemples:
-<h1></h1>, <h2></h2>, … <h6></h6> : insert un titre du plus important au moin important
-<img/> : permet d'ajouter une image
-<a> : permet d'ajouter un lien
-<p> : designe un paragraphe
-<strong> : donne de l'importance au texte et le met en majuscule
-<i> : italique
-<em> : italique
-<br/> : ajoute un retour à la ligne
-<ul> : liste non ordonée
-<ol> : liste pordonée
-<li> : list element
